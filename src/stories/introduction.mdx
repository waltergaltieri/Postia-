# Sistema de Diseño Postia

Bienvenido al sistema de diseño de Postia, una plataforma SaaS premium para agencias de marketing que necesitan generar contenido de alta calidad de manera eficiente.

## Filosofía de Diseño

Nuestro sistema de diseño se basa en tres principios fundamentales:

### 1. **Estética Premium Primero**
Cada elemento visual debe transmitir sofisticación y profesionalidad, compitiendo con herramientas de clase mundial como Figma, Linear y Notion.

### 2. **Contenido como Protagonista**
La interfaz debe realzar, no competir con el contenido generado por IA. Utilizamos fondos neutros y espaciado generoso para hacer destacar el trabajo creativo.

### 3. **Eficiencia Contextual**
Optimizamos cada interacción para los flujos de trabajo específicos de agencias de marketing, priorizando velocidad y comodidad para usuarios expertos.

## Tokens de Diseño

### Colores
- **Primarios**: Azul sofisticado inspirado en herramientas premium
- **Neutros**: Grises cálidos que proporcionan elegancia sin competir con el contenido
- **Estados**: Colores claros y distintivos para feedback y estados del sistema

### Tipografía
- **Fuente Principal**: Inter - Sans-serif moderna y legible
- **Fuente Monospace**: Geist Mono - Para código y datos técnicos
- **Escala**: Sistema basado en proporciones armónicas

### Espaciado
- **Sistema Base**: 8px como unidad fundamental
- **Espaciado Generoso**: Priorizamos el espacio en blanco para elegancia
- **Jerarquía Clara**: Diferentes niveles de espaciado para crear estructura visual

## Componentes

### Componentes Base (Atoms)
- **Button**: Botones premium con micro-interacciones
- **Badge**: Indicadores de estado y categorías
- **Input**: Campos de entrada elegantes y funcionales

### Componentes Compuestos (Molecules)
- **Card**: Contenedores versátiles con efectos de elevación
- **ContentCard**: Especializado para contenido generado por IA
- **NavigationItem**: Elementos de navegación contextual

### Componentes Complejos (Organisms)
- **ContentGrid**: Grid masonry para contenido visual
- **NavigationSidebar**: Navegación principal con branding dinámico
- **AIGenerationWorkflow**: Interfaz de generación de contenido

## Patrones de Interacción

### Micro-interacciones
- **Hover Effects**: Elevación sutil y cambios de color
- **Loading States**: Skeleton screens y animaciones de progreso
- **Feedback**: Toasts elegantes y confirmaciones visuales

### Animaciones
- **Transiciones Suaves**: 300ms para cambios de estado
- **Animaciones de Entrada**: FadeIn, SlideIn para nuevo contenido
- **Celebraciones**: Micro-animaciones para acciones exitosas

## Accesibilidad

Cumplimos con los estándares WCAG 2.1 AA:
- **Contraste**: Mínimo 4.5:1 para texto normal
- **Navegación por Teclado**: Soporte completo con indicadores de foco
- **Lectores de Pantalla**: Etiquetas ARIA y estructura semántica

## Responsive Design

### Breakpoints
- **Mobile**: < 768px - Navegación táctil optimizada
- **Tablet**: 768px - 1024px - Layouts adaptativos
- **Desktop**: > 1024px - Experiencia completa con atajos

### Adaptaciones
- **Móvil**: Bottom navigation, gestos swipe, previews optimizados
- **Tablet**: Sidebar colapsible, multi-touch, modo landscape
- **Desktop**: Sidebar fijo, atajos de teclado, multi-panel
